# 🏫 Veltrio – Modern Real Estate Web Application

## 🔗 **Live Website:** https://veltrio-real-estate-website.vercel.app/

Veltrio is a modern, scalable, and fully responsive real estate web application built with React. The platform provides an intuitive browsing experience for property listings, advanced filtering options, agent interaction, and content-driven pages such as blogs and services. It is designed with performance, accessibility, and clean UI architecture in mind.


## ✨ Key Features

### 📱 Responsive Design
- Fully responsive layout for mobile, tablet, and desktop devices
- Adaptive UI components for optimal viewing on all screen sizes
- Mobile-friendly left-side slide menu navigation


### 🎨 Theme Management
- Supports Light, Dark, and System themes
- Enhances user accessibility and visual comfort
- Seamless theme switching without page reload

### 🏘️ Property Management
- Card-based real estate property listings
- Individual property details page with complete information
- Smooth navigation between listing and details view
- Integrated Contact Agent section for direct inquiries

### 🔍 Advanced Filtering & Sorting

- Filter properties based on:
    - Category
    - City
    - Property Type
    - Number of Bedrooms
    - Number of Bathrooms
- Sort properties by:
    - Price (Low to High)
    - Price (High to Low)
 
### 👥 Real Estate Agents
- Agent profiles displayed in a structured, card-based layout
- Clear presentation of agent information for client trust

### 📊 Informational Pages
- Services page explaining business offerings
- Statistics section highlighting achievements and data
- About page describing the platform and vision
- “Why Choose Us” section to build credibility

### 📰 Blog System
- Blog posts displayed in a card-based layout
- Dedicated blog details page
- Popular blogs section for better content engagement


### 📍 Contact & Footer
- Fully functional **Contact page** with integrated map.
- Modern footer with essential quick links and school information.
